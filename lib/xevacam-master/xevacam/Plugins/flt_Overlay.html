Image filter documentation: Overlay

Overlay Programming name
Programming name to be used in Xeneth GUI application settings "Auto start filters" or when queueing the filter while programming with Xeneth SDK.
Overlay Overlay [Read/Write]
Location of the (.png) file to be copied in the overlay of the frame.
  Value: c-string
Position Position [Read/Write]
Position of the image with respect to the frame.
  Values: Top left(0),Top right(1),Bottom left(2),Bottom right(3),Center(4)
