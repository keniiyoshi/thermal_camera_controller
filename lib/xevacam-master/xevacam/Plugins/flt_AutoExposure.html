Image filter documentation: Auto exposure

AutoExposure Programming name
Programming name to be used in Xeneth GUI application settings "Auto start filters" or when queueing the filter while programming with Xeneth SDK.
IntegrationTimeMin Minimal integration time [Read/Write]
Lower bound on the integration times used.
  Value: Positive number
IntegrationTimeMax Maximal integration time [Read/Write]
Upper bound on the integration times used.
  Value: Positive number
Mode Mode [Read/Write]
Select whether either the average image value or the top of the histogram is steered to the Target value.
  Values: Top(0), Average(1).
Target Target value (%) [Read/Write]
Depending on the selected Mode either the average image value or the top of the histogram is steered to the Target value.
  Values: floating number in [0,100].
Tolerance Tolerance (%) [Read/Write]
As soon as the difference between the current image value and the Target value is larger than the Tolerance, a new integration time is searched for.
  Values: floating number in [0,100].
Outliers Outliers (%) [Read/Write]
Outliers defined as percentage of the total amount op pixels.
  Values: floating number in [0,100].
