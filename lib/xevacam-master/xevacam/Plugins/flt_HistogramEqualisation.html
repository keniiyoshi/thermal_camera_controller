Image filter documentation: Histogram equalisation

HistogramEqualisation Programming name
Programming name to be used in Xeneth GUI application settings "Auto start filters" or when queueing the filter while programming with Xeneth SDK.
Not available on Linux.
HistogramEqualisationMode Mode [Read/Write]
In Regions mode the image is subdivided in regions according to X Regions and Y Regions. The map used to transform a pixel depends on the local histogram. In the Window mode the image will only be optimized in the window defined by Offset X, Offset Y, Width and Height.
  Values: Regions(0), Window(1)
HistogramEqualisationClipLimit Clip limit [Read/Write]
Defines the strength of the contrast enhancement.
  Values: number in [0,100]
HistogramEqualisationXRegions X regions [Read/Write]
When Mode is Regions, the image is divided into equally sized regions. The width of each of these regions is the image width divided by X regions.
  Values: integer in [1,8]
HistogramEqualisationYRegions Y regions [Read/Write]
When Mode is Regions, the image is divided into equally sized regions. The height of each of these regions is the image height divided by Y regions.
  Values: integer in [1,8]
WindowOffsetX Offset X [Read/Write]
When Mode is Window, pixels in the window defined by Offset X, Offset Y, Width and Height are taken into account to define the image transformation.
  Values: integer in [0,width of the image - 1]
WindowOffsetY Offset Y [Read/Write]
When Mode is Window, pixels in the window defined by Offset X, Offset Y, Width and Height are taken into account to define the image transformation.
  Values: integer in [0,height of the image - 1]
WindowWidth Width [Read/Write]
When Mode is Window, pixels in the window defined by Offset X, Offset Y, Width and Height are taken into account to define the image transformation.
  Values: integer in [1,width of the image]
WindowHeight Height [Read/Write]
When Mode is Window, pixels in the window defined by Offset X, Offset Y, Width and Height are taken into account to define the image transformation.
  Values: integer in [1,height of the image]
