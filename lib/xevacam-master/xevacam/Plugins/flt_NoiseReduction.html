Image filter documentation: Noise reduction

NoiseReduction NoiseReduction
Programming name to be used in Xeneth GUI application settings "Auto start filters" or when queueing the filter while programming with Xeneth SDK.
Not available on Linux.
numberofiterations Filter size [Read/Write]
Radius of the region around a pixel taken into account for the smoothing operation.
  Values: integer in [1,16]
localcontrast Contrast (%) [Read/Write]
Defines how much differences between neighboring pixels are reduced. The maximal (minimal) contrast is obtained at value 1 (100).
  Values: integer in [1,100]
