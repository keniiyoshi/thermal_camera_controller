Image filter documentation: Smoothing

Smoothing Programming name
Programming name to be used in Xeneth GUI application settings "Auto start filters" or when queueing the filter while programming with Xeneth SDK.
MatSizeX Matrix size X [Read/Write]
The filter replaces every pixel by an average value of the pixel and its neighbouring pixels. The pixels taken into account are those within the rectangle defined by Matrix size X and Matrix size Y.
  Values: odd integer in larger than 3
MatSizeY Matrix size Y [Read/Write]
The filter replaces every pixel by an average value of the pixel and its neighbouring pixels. The pixels taken into account are those within the rectangle defined by Matrix size X and Matrix size Y.
  Values: odd integer in larger than 3
