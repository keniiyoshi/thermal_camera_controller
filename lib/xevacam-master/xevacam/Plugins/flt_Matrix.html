Image filter documentation: Matrix

Matrix Programming name
Programming name to be used in Xeneth GUI application settings "Auto start filters" or when queueing the filter while programming with Xeneth SDK.
Matrix Matrix [Read/Write]
The n^2 entries of a n x n matrix that defines a convolution filter.
For example: "0;0;0;0;1;0;0;0;0" for a 3 times 3 matrix.
  Value: c-string of semi-colon separated numbers.
Divisor Divisor [Read/Write]
Each matrix element is divided by the divisor.
  Values: positive number.
Offset Offset [Read/Write]
Number added to each matrix element.
  Values: number.
